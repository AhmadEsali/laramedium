# Lara Medium

## Steps

```bash
copy .env.example to .env and add database info
```

```bash
composer install
```

```bash
npm install
```

```bash
npm run dev

```

```bash
php artisan migrate
```

```bash
php artisan db:seed
```

```bash
php artisan serve
```

Then go to `http://localhost:8000`

dashboard link `http://localhost:8000/dashboard`
